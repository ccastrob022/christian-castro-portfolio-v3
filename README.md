# Portfolio (Static Site) — Christian Castro

This is a lightweight static portfolio site (HTML/CSS/JS) ready to upload to any hosting provider.

## Deploy (any hosting)
Upload these files/folders to your hosting public directory (often `public_html/`):
- index.html
- style.css
- script.js
- assets/

## Update
- Replace `assets/Christian_Castro_Resume_2026.pdf` with your latest PDF resume (keep the same filename).
- Replace `assets/og-image.png` with your preferred social preview image (1200x630 recommended).
- Update contact links inside `index.html` if needed.

## Optional: Add your LinkedIn / GitHub
Search in `index.html` for the Contact section and add links.
